# Изменения 11.05.2022

Обновлена версия API до 1.1, документация ведется только для последней версии, в версии 1.0 не будет ломающих изменений

## Изменения в версии API 1.0

1. Метод `/search` ([Поиск туров](../Поиск туров.md))
    1. Добавлено поле `accommodation`
    2. Добавлен список фотографий тура - поле `images`

## Изменения в версии API 1.1

Нужно передавать параметр `v=1.1`

1. Метод `/search` ([Поиск туров](../Поиск туров.md))
    1. Удалено поле `livings`
    2. Добавлено поле `accommodation`
    3. Добавлен список фотографий тура - поле `images`
2. Метод `/list/calendar-collections` переименован в `/list/calendar-presets` для лучшего отображения смысла этой
   сущности ([Календарные пресеты](../Календарные пресеты.md))
3. Изменен формат выдачи метода `/list/calendar-presets` ([Календарные пресеты](../Календарные пресеты.md))
